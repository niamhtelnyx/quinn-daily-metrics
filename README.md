# meeting-sync

Auto-SFDC Post-Meeting Sync — captures meeting transcripts, extracts structured deal data, and pushes to Salesforce with human-in-the-loop validation.

## Problem

Sales reps don't update Salesforce after meetings. Notes, context, next steps, and deal signals get lost — degrading pipeline accuracy and forecast reliability.

## Solution

Automated pipeline: Fellow transcript → AI extraction → SFDC match → Slack approval → CRM update.
